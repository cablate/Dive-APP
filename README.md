## Quick Start

1. Install required tools:

   - Flutter SDK
   - Dart SDK
   - Android Studio or VS Code
   - iOS development requires MacOS and Xcode

2. Clone the project:

   ```
   git clone https://github.com/cablate/Dive-APP.git
   ```

3. Install dependencies:

   ```
   flutter pub get
   ```

4. Launch the app:
   ```
   flutter run
   ```
